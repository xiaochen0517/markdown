# 定位

```
定位（position）
- 定位是一种高级的布局方式
- 通过定位可以将元素摆放到页面的任意位置
- 可选值：
	- static：默认值，元素没有开启定位
	- relative：相对定位
	- absolute：绝对定位
	- sticky：粘滞定位
```

## 1、相对定位

```
- 特点：
	- 元素开启相对定位时不设置偏移量，元素不会发生任何变化。
	- 定位原点是元素在文档流中的位置
	- 相对定位会提升元素的层级
	- 相对定位不会使元素脱离文档流
	- 相对定位不会改变元素的性质
	
- 偏移量（offset）：
	- 当元素开启定位后，可以使用偏移量来设置元素的位置
	- top：定位元素和定位位置上方的距离
	- bottom：定位元素和定位位置下方的距离
	- left：定位元素和定位位置左侧的距离
	- right：定位元素和定位位置右侧的距离
```

## 2、绝对定位

```
- 特点：
	- 不设置偏移量时元素的位置不会发生变化
	- 元素会从文档流中脱离
	- 会改变元素的性质
	- 会将元素提升一个层级
	- 元素是相对于其包含块进行定位的

- 包含块：
	- 正常情况下，包含块就是离当前元素的最近的父块元素
	- 绝对定位下，包含块就是离当前元素最近的开启定位的父元素
		- 若所有的父元素都没有开启定位，就相对于根元素进行定位
		- html（根元素，初始包含块）
```

## 3、固定定位

```
- 特点：
	— 固定定位的绝大多数特点都与绝对定位相同
	- 固定定位永远参照于浏览器的视口进行定位
	- 固定定位不会随着网页的滚动条滚动
```

## 4、粘滞定位

```
- 特点：
	- 粘滞定位可以在元素到达某个位置时将其固定
```
