# 数据库引擎

> INNODB：默认使用
>
> MYISAM：早期版本默认使用

|                | MYISAM | INNODB          |
| -------------- | ------ | --------------- |
| 支持事务       | ✘      | ✔               |
| 支持数据行锁定 | ✘      | ✔               |
| 支持外键约束   | ✘      | ✔               |
| 支持全文索引   | ✔      | ✔（仅支持英文） |
| 表空间大小     | 较小   | 较大            |

特点

- MYISAM：节约空间，速度较快
- INNODB：安全性高，支持事务处理，多表多用户操作
